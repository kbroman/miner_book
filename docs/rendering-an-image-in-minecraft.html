8 Rendering an image in Minecraft

The miner package makes it possible to manipulate minecraft. Here I will illustrate one use of this: reading in an image file and rendering it within minecraft as blocks.

8.1 Load and prepare an image file

First, letâs download an image file from the web. Weâll load it directly into R from the web, using the imager package.

              "/300x300/public/karthik_ram.jpeg?itok=loiSP_tm")
img <- load.image(url)

Print to see the size of the image.

## Image. Width: 300 pix Height: 300 pix Depth: 1 Colour channels: 3

The image is stored as a 4-dimensional array: horizontal and vertical position, time, and color.

## [1] 300 300   1   3

Letâs convert it to grayscale, and make it 100x100.
